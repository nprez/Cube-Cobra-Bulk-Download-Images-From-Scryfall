# Cube-Cobra-Bulk-Download-Images-From-Scryfall
Python script that bulk downloads scryfall card images based on cube cobra export for use in tabletop simulator.

python bulkDownload.py [cube_id]

Collate these images using tabletop simulator's deck builder tool located at C:\Program Files (x86)\Steam\steamapps\common\Tabletop Simulator\Modding\Deck Builder
